# 🔳 Gerador de QR Code

Projeto desenvolvido como parte dos meus estudos em desenvolvimento web, com foco na prática de JavaScript, manipulação do DOM e integração com APIs externas.

## 📖 Sobre o Projeto

Esta aplicação permite gerar QR Codes a partir de textos ou links informados pelo usuário de forma simples e rápida.

O projeto foi construído durante meu processo de aprendizado em front-end, servindo como prática para consolidar conceitos fundamentais do desenvolvimento web e compreender a integração entre interfaces e serviços externos.

## 🚀 Funcionalidades

* Geração dinâmica de QR Codes
* Interface simples e intuitiva
* Validação de entrada do usuário
* Atualização em tempo real
* Layout responsivo

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* API/Biblioteca de geração de QR Code

## 💡 Conceitos Praticados

* Manipulação do DOM
* Eventos JavaScript
* Consumo de APIs
* Validação de dados
* Estruturação de aplicações front-end
* Organização de código

## 🎯 Objetivo

Este projeto foi desenvolvido com fins educacionais para praticar conceitos de desenvolvimento front-end e ampliar minha experiência com JavaScript na construção de aplicações interativas.
